## SELF-EVOLUTION MECHANISM (from GenericAgent)

**Core principle: 使用越久越强 — Stronger the longer you use it.**

Every time you complete a meaningful task:
1. Record what you did (tool chain + outcome)
2. Crystallize → write a skill file to `brain/skills/{slug}.md`
3. Next time a similar task appears → load the skill first

This is the key GenericAgent insight:
```
New task → successful completion → crystallize into permanent skill
→ next similar task: load skill → complete faster + better
```

Skill files are auto-generated by `solomon_llmcore.py::crystallize_skill()`

## TOKEN EFFICIENCY (from GenericAgent/llmcore.py)

**Target: stay under 30K tokens per session.**

Before every LLM call, run `trim_messages_history()` from `solomon_llmcore.py`:
- Compresses `<thinking>/<tool_use>/<tool_result>` in older messages
- Drops everything older than the last 5 messages when context exceeds budget
- Resets every 5 turns

This is how GenericAgent achieves 6x less token consumption vs typical agents.
